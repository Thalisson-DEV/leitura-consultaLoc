Como usar os botões da interface

Nossa interface possui botões especiais para melhorar sua experiência. Confira abaixo:

Botão de Menu

Localizado no canto inferior direito em dispositivos móveis. Use-o para abrir o menu de navegação lateral.

Botão de Tema

Localizado no canto superior direito. Alterna entre tema claro e escuro para melhorar sua experiência visual.
